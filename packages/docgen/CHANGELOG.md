# @trysquare/docgen

## 0.2.0

Initial release. Generates Doxygen-style API reference documentation from verified test chains. Implements the reporter interface (`onSuiteEnd`). Supports `types: 'infer' | 'explicit' | 'typescript'` and `format: 'html' | 'markdown' | 'both'`. TypeScript source integration is stubbed for a future release.
